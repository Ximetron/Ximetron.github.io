Módulo 1: Configuración de Windows Defender

Progreso:

1.1 Activación de la protección en tiempo real

Windows Defender es la solución antivirus integrada en Windows 11. Para asegurarte de que está correctamente configurado, sigue estos pasos:

  • Abre la Configuración de Windows (tecla Windows + I)
  • Selecciona "Privacidad y seguridad"
  • Haz clic en "Seguridad de Windows"
  • Selecciona "Protección contra virus y amenazas"

Consejo: Verifica que la opción "Protección en tiempo real" esté activada para una protección continua contra malware.
